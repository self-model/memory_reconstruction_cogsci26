# Symmetry experiment data

Raw JATOS output from Experiment 1 (symmetry grid game), collected in two Prolific batches.

**Note:** The `hover_log` column has been removed from the JATOS CSV files before committing. It contained continuous mouse-hover tracking data and inflated the raw files to ~27MB (batch 1) and ~145MB (batch 2), exceeding GitHub's file size limit. The full original files are retained in the source repository (`rememberingRound1`).
